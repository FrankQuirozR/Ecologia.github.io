Introduccion

R y Rstudio

R es un lenguaje de programación y un entorno de desarrollo diseñado específicamente para análisis estadístico y gráfico. Es de código abierto y cuenta con una amplia gama de paquetes y librerías que permiten a los usuarios realizar diversas operaciones estadísticas, modelado de datos, visualización y más. RStudio, por otro lado, es un entorno de desarrollo integrado (IDE) para R. Proporciona una interfaz amigable para trabajar con R, que incluye características como resaltado de sintaxis, depuración de código, administración de paquetes y visualización de resultados.

Global Biodiversity Information Facility (GBIF)

Es una organización internacional que proporciona acceso gratuito y abierto a datos sobre la biodiversidad de todo el mundo. GBIF recopila, estandariza y distribuye datos de biodiversidad, tales como registros de especies, observaciones de la naturaleza, colecciones de museos, y otros tipos de información relacionada con la vida silvestre. El objetivo principal de GBIF es facilitar el acceso a la información sobre la biodiversidad para apoyar la investigación científica, la toma de decisiones en la conservación, el monitoreo ambiental y la educación.

Datos

Los datos utilizados se descargaron de GBIF, una red internacional e infraestructura de datos de acceso abierto que ofrece información de biodiversidad de las formas de vida en la tierra.

Para acceder a los datos:

  1. Acceda a GBIF con una cuenta de usuario.
  2. Diríjase a la sección de Datos y haga clic en Registro de Presencia.
  3. En el buscador, introduzca el nombre científico de la especie, en este caso, el Cormorán Guanay (Phalacrocorax bougainvillii).
  4. Agregue un filtro, en este caso, Perú.
  5. En el Visor del Registro de Presencia, haga clic en el botón de descarga (.csv).

Importación

Lectura de Datos CSV

Se hace la lectura del archivo .csv con read.csv(), el cual contiene argumentos de la ruta del archivo y el tabulador como separador de las columnas representado por "\t".

Previamente se debe hacer llamado a las siguientes librerías:

  • library(tidyverse)
  • library(sf)
  • library(ggplot)
  • library(ggplot2)
  • library(RColorBrewer)

Se tipean las siguientes líneas de código:

Paujil <- read.csv("C:/Users/frank/Desktop/PAUJIL/data_paujil.csv",sep="\t")


Visualización de los Datos CSV

Se utilizará el comando view() que permite visualizar los datos en una tabla en Rstudio.

Para saber las dimensiones de los datos en números de filas y columnas usamos el comando dim():

dim(Paujil)

Para saber de manera ordenada los nombres de los campos de los datos se usa el comando names() y sort():

sort(names(Paujil))

Para mostrar los datos sin repetir los elementos de una columna específica se usa el comando unique():

unique(Paujil$countryCode)

Para filtrar los datos guardándolos en una variable se usa el comando filter():

Paujil_filtrado <- filter(Paujil, Paujil$countryCode == "PE")

dim(Paujil_filtrado)

Para recortar los datos en función de una selección se usa el comando subset():

Paujil_cortado <- subset(Paujil_filtrado, !is.na(decimalLongitude))

dim(Paujil_cortado)

Por último, exportamos los datos usando el comando write.csv():

write.csv(Paujil, "C:/Users/frank/Desktop/PAUJIL/coordenada_data_paujil.csv")
